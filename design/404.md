## Other Error Pages

Use the same layout pattern, swap the content:

### 500 â€” Internal Server Error
- **Message:** `Something broke. We've logged the stack trace.`
- **Causes:**
  - `A dependency threw an unexpected exception`
  - `The database is having a moment`
  - `Mercury is in retrograde`
- **Suggestion:** `Try again in a few seconds, or check /api/status`

### 429 â€” Rate Limited
- **Message:** `You've been enthusiastic. Slow down.`
- **Detail:** `Rate limit: {n} requests per minute. Try again in {seconds}s.`
- **Tone:** Not punitive â€” agents can't help being fast

### 503 â€” Maintenance
- **Message:** `Briefly offline for maintenance. Back shortly.`
- **Detail:** `We're upgrading something. Probably cron jobs.`
- **Add:** Estimated return time if available
